Business Decisions

Comments

We aim to shortlist an ideal number of key factors to explain a significant proportion of the variance in the data. This would reduce the dimensionality of the survey responses such that it is easier to perform further analysis of the customer population in a more manageable manner.

Our Approach

Factor Analysis in 6 steps

  1. Confirm the data in metric

  2. Decide whether to scale or standardize the data

  3. Check the correlation matrix to see if Factor Analysis makes sense

  4. Develop a scree plot and decide on the number of factors to be derived

  5. Interpret the factors (consider factor rotations - technical but useful)

  6. Save factor scores for subsequent analyses

Let's follow these steps.

Step 4. Develop a scree plot and decide on the number of factors to be derived

Comments

We can observe from the data table that 52.1% of variance can be explained with 5 variables. Beyond the 5th variable, additional variance explained increases only marginally with each component added.




Comments

We can observe from the screen plot that the elbow bends after the fifth components which is aligned with the data table analysis.
